# ModernTech Solutions â€“ Payroll Management System


A **web-based HR management system** designed for **ModernTech Solutions**, a software development company specializing in healthcare solutions. This project addresses the company's pain points by centralizing employee data, automating payroll, and streamlining HR processes.

### **Pain Points**
1. **Employee data** stored in multiple Excel files.
2. **Time-off requests** managed via emails and shared calendars.
3. **Payroll calculations** done manually with complex spreadsheets.
4. **Performance reviews** stored as Word files in shared drives.

## ðŸ“Œ Project Overview

The **Payroll Management System** is a web application that helps organizations manage employee compensation, generate payslips, and track payroll data. It includes features like:

- **Employee Payroll Calculation:** Calculate salaries based on hourly rates, base salary, and deductions.
- **Payslip Generation:** Generate and download payslips as PDFs.
- **Data Visualization:** View payroll analytics with **Chart.js**.
- **Responsive UI:** Built with **Vuetify** for a seamless user experience.
